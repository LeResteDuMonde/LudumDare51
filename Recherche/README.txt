- CUSTOM STENCILS -

In the desktop version you can upload your own stencils to the game

FORMAT

They must follow a this format :
- size : 1080 * 1080
- extension : .png

UPLOAD FOLDER PATH

The game must have been lauched one for the folder to exist
You can usualy find it under :

user/AddData/Roaming/Godot/app_userdata/[project_name]

POINTS SYSTEM IN GAME

No matter what in game points will be calculate as follow :
- transparent pixel shouldn't be drawn over
- semi-transparent pixels or opaque pixels should be drawn over

The color of the pixel as no influence over the algorithm
